* html是网页的结构
* css是网页的外观
* JS是网页的行为

* 只学习前端技术做出来的是静态网站
  * 静态网站唯一的功能是供用户浏览而不能与服务器进行交互。

---

* 推荐的前端学习路线
  * `HTML → CSS → JavaScript → jQuery → CSS3 → HTML5 → Vue.js`
* 初学者不要想着精通了某一门技术再去精通另一门技术，Web领域不少技术之间都存在交叉关系
* （不一定要把html精通了再去学CSS，这也不太可能做到）

---

* html学习
  * 什么是html
    * 不是一门编程语言
    * 描述性的标记语言
    * 是网页的标准语言
  * 基本语法
    * `<标签符>内容</标签符>`
  * 如何学习html
    * 学习各种标签
      * 有时我们也把标签说成元素
    * 不要图快，尽量把每一个概念都理解清楚
    * 学习HTML不需要任何编程基础
    * 掌握html后可以做一个简单网页
    * html入门最多只需要一周，但还要学习html进阶，不然并不具备实际开发能力
    * 没有HTML基础是学不来HTML5的
      * 因此要先学HTML
